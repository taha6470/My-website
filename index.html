Languages

HTML

HTML (HyperText Markup Language) is a markup language used for creating web pages and applications. I have used html to make many webpages and forms. It is the skeleton of a website, It has no design and is in basic font (serif).

CSS

CSS (Cascading Style Sheets) is a stylesheet language used for describing the presentation of a document written in HTML. It is used to add styling and layout to web pages. In this website you can view the CSS and html using the inspect element on your browser. CSS can make any webpage visually appealing and easy to read.

JavaScript

JavaScript is a high-level, interpreted programming language. It can make the webpage more interactive and easier to use. javascript is also used to make web applications, mobile apps, games and many more!
